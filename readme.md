# House Of CSE


Alien's Photography  is a personal website where you can get different services.

## Features

- Different Types of Services
- Secure login
- Register options
- Data loaded from API
- Attractive UI
- Build with new technologies etc.
- Personal Database

## Technology,Libraries and Language Used

- HTML
- CSS
- JAVA SCRIPT
- Tailwind CSS
- React.js
- Node.js
- MongoDB

## Login System Implemented

- Log in with Email and Password
- Log in with Google Popup
- Log in with Github
- JWT authentication
